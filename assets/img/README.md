# Profile photo

Drop your headshot here as **`profile.jpg`** (this exact path: `assets/img/profile.jpg`).

- Square image works best (it's cropped to a circle). ~320×320px or larger.
- `.jpg`, `.png`, or `.webp` all work — if you use a different extension, update
  the `src` in `index.html` (the `<img>` inside `.hero__photo`).

Until the file exists, the site automatically shows a clean **"CY"** monogram
in its place, so nothing looks broken.
